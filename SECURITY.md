# Security Policy

## Supported Versions

This is a portfolio workflow template. Security updates are handled on the `main` branch.

## Reporting a Vulnerability

Open a GitHub issue with reproduction steps and the affected workflow version.

Do not commit:

- OpenRouter or OpenAI API keys
- Google OAuth tokens or credential exports
- private Google Sheet IDs or URLs
- customer, supplier, or inventory data from a real business
- n8n environment files or local database files
