# Contributing

Thanks for your interest in improving this workflow.

## Local Checks

```bash
npm run validate
```

The validator uses only built-in Node.js modules.

## Workflow Guidelines

- Keep exported workflows sanitized before committing.
- Do not commit real credential IDs, API keys, OAuth data, or private sheet URLs.
- Document any new node, credential, sheet column, or chat command behavior.
- Keep the workflow focused on inventory check and update tasks.
